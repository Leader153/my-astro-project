# План создания веб-бота для aflagot-haifa.com 🤖⛵

В этом документе мы подробно распишем, как мы создадим умного помощника (чат-бота) специально для нашего сайта, который будет консультировать клиентов по яхтам круглосуточно.

Этот бот будет **максимально лёгким, быстрым и независимым** от старого проекта с Twilio. Он будет работать только в текстовом виде на сайте.

---

## 🏗️ Как работает система (Архитектура)

Система состоит из двух частей:

1. **Фронтенд (Окошко на сайте Astro)**
   - Это красивый пузырек чата в нижнем углу экрана (HTML + CSS + простой JavaScript).
   - Когда клиент набирает сообщение и жмёт "Отправить", JavaScript перехватывает этот текст и по невидимому каналу (через `fetch`) отправляет его на наш _Бэкенд_.
2. **Бэкенд (Маленький сервер-мозг)**
   - Это отдельная программа на Node.js (буквально 1-2 файла), которая будет работать на твоём VPS-сервере.
   - Её единственная задача: получить сообщение от клиента с сайта -> отправить его в нейросеть Google Gemini (вместе с инструкцией: "Ты продавец яхт в Хайфе, вот наши цены...") -> получить ответ от ИИ -> вернуть этот ответ обратно на сайт.

> **Почему мы не можем отправлять запросы в Gemini прямо с сайта?**
> Ключ от Gemini API — это пароль к твоему кошельку в Google. Если мы впишем его прямо в код сайта, любой хакер сможет нажать F12 (просмотр кода браузера), украсть твой ключ и использовать нейросеть за твой счёт. Поэтому ключ должен лежать только на нашем Бэкенд-сервере, куда ни у кого нет доступа.

---

## 🙋‍♂️ Твоя задача (Что нужно сделать тебе ДО начала кода)

Тебе нужно зарегистрироваться и получить **API Ключ** от Google Gemini. Это мозг нашего бота.

**Шаги для тебя:**

1. Зайди на сайт **Google AI Studio**: [https://aistudio.google.com/](https://aistudio.google.com/)
2. Войди под своим Google-аккаунтом.
3. В левом меню найди и нажми кнопку **"Get API key"** (Получить API ключ).
4. Нажми **"Create API key"** (Создать API ключ). Google может попросить выбрать или создать Google Cloud Project — просто соглашайся.
5. Когда ключ сгенерируется (это будет длинная строка, похожая на пароль), **скопируй его и сохрани себе в надёжное место!** (Сюда в чат мне его тоже не кидай в сыром виде, это секрет).
6. _Примечание:_ Базовое использование (до 15 запросов в минуту) на момент написания **абсолютно бесплатно**.

## 🛠️ Наш план действий (Что буду делать я, Сайман)

Как только у тебя будет готов API-ключ, скажи мне, и мы начнём работу по этим шагам:

### Этап 1: Создание Фронтенда (Окошка на сайте)

1. В папке `src/components` я создам файл `ChatWidget.astro`.
2. Я напишу для него классный дизайн (чтобы он выглядел современно, в стиле твоего сайта, открывался по клику и красиво анимировался).
3. Добавлю JavaScript логику внутри виджета, чтобы он умел отправлять сообщения на пока ещё несуществующий локальный сервер (например, на `http://localhost:3000/api/chat`).
4. Мы добавим этот виджет в `Layout.astro`, и ты сможешь посмотреть на него на своём компьютере (`npm run dev`).

### Этап 2: Создание Бэкенда (Сервера-посредника)

В папке с твоими проектами (можно прямо рядом с папкой `astro`, например `c:\a my box\simple-bot-server`) мы создадим новый мини-проект.

1. Я напишу команду для инициализации (создания `package.json`) и установки всего двух нужных библиотек (`express` для создания сервера и `@google/generative-ai` для связи с Gemini).
2. Напишу код сервера `server.js` (~50 строк), который будет:
   - Хранить секретный системный промпт ("Ты продавец яхт, вот прайс: девичник 1500₪...").
   - Принимать POST-запросы от виджета чата.
   - Опрашивать Gemini и возвращать чистый текст ответа.
3. Ты запустишь этот сервер у себя на компьютере и мы протестируем весь цикл: "Пишешь в чат на сайте -> Идёт запрос на локальный сервер -> Сервер идёт в Google -> Ответ возвращается на сайт".

### Этап 3: Деплой (Запуск на живом сайте)

Когда всё будет идеально работать на твоём компьютере:

1. Мы загрузим код маленького сервера на твой VPS (194.5.159.34).
2. Настроим на VPS программу `PM2`, чтобы сервер бота работал там вечно (и перезапускался при сбоях).
3. Настроим Nginx так, чтобы запросы по адресу `https://aflagot-haifa.com/api/chat` летели прямо в наш сервер-бот.
4. Сделаем Commit & Push фронтенда — и бот оживёт для реальных клиентов!
